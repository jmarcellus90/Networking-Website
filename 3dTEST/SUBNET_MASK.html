Subnet Mask

Is a 32-bit (X.X.X.X) combination used to describe which portion of an address refers to the network address and which part refers to the host address.Subnet Mask is made by setting network bits to all "1"s and setting host bits to all "0"s. With our demonstration, we will be using a class C address. This class subnet mask looks like this: 255.255.255.0 or (11111111.11111111.11111111.00000000). This address will help determine what will be the network address for this excerise.
